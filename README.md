# 使用方式
- 克隆仓库到本地，然后安装```requirements.txt```，然后```python app.py```即可运行
- 或是下载打包好的文件，解压缩运行
# 运行环境
- CUDA 12.1 (与 spconv 强绑定，如果你使用的是比如 CUDA11.8 ，则安装 spconv-cu118 ， torch 等同理）
- Python 11.3
# 已知问题
- 在三维生命游戏中，过大的存活细胞数量会导致程序崩溃（比如 30000 个）
- 我已提供了一些预设和画笔模式，但是我还没有添加在不同方位摆放预设的功能（但你可以修改```presets.py```里的center实现修改预设生成的位置）
- 现有的打包文件启动时长较慢，如果有兴趣可以尝试优化一下
